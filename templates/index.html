1. Kaggle

Cari dataset Tokopedia Product Reviews dan simpan file CSV ke folder data/.

2. ambil_data.py

Membaca dataset, menampilkan info kolom, dan menghasilkan grafik awal.

3. cnn.py

Preprocessing, tokenisasi, padding, training CNN, dan evaluasi model.

4. Flask Web

Tampilkan grafik, metrik, dan prediksi teks baru di halaman hasil.

Alur Praktikum

  1. Jelajahi Kaggle dan pahami struktur dataset review.
  2. Unduh atau salin CSV ke folder data/.
  3. Jalankan ambil_data.py untuk melihat isi data dan grafik awal.
  4. Jalankan cnn.py untuk membersihkan teks, melatih model, dan menyimpan visualisasi.
  5. Buka app.py untuk menampilkan hasil dan mencoba prediksi kalimat baru.

Contoh Perintah

pip install -r requirements.txt
python ambil_data.py --download
python cnn.py
python app.py
